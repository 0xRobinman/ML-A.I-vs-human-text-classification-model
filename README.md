## A.I vs Human text classification model


# Getting started

``` bash
git clone https://github.com/0xRobinman/ML-A.I-vs-human-text-classification-model.git
cd ML-A.I-vs-human-text-classification-model
pip install -r requirements.txt
python3  model.py
```
